<h1 class="text-red" align="center">Projeto RocketSeat Página Dinâmica</h1>

<h2>Como executar este projeto ? ⚙️</h2> <br>

<p align="center">Para executar esse projeto é necessário baixar todos os arquivos e diretórios, dar o comando npm start no seu terminal e acessar a porta 5000 local.</p>

<p>Colocar em prática os conhecimentos adquiridos no curso da Rocketseaat do módulo 3, que mostra como tornar uma página estática em uma página dinâmica <p>
